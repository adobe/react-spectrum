# React Spectrum Native Starter

Expo smoke app for `@react-spectrum/native`.

This starter verifies that the workspace package can be imported by an Expo app and that the current native surface renders together under `Provider`, including overlays and representative collection components.

## Run

```sh
yarn install
yarn start
```

The starter is intentionally small. It is not a design-system documentation site; it is a packaging and interaction smoke target for the native package.
