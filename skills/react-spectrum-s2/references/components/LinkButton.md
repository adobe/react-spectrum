# Link

Button

A LinkButton combines the functionality of a link with the appearance of a button. Useful for allowing users to navigate to another page.

```tsx
import {LinkButton} from '@react-spectrum/s2';

<LinkButton />
```

## A

PI

```tsx
<LinkButton>
  <Icon />
  <Text />
</LinkButton>
```
