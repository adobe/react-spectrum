# Components

This directory contains the public component surface for `@react-spectrum/native`.

Implemented groups:

- Buttons: `Button`, `ActionButton`, `ToggleButton`, `ButtonText`
- Text and layout: `Text`, `Heading`, `Flex`, `View`, `Divider`, `Badge`, `StatusLight`
- Fields and toggles: `Field`, `TextField`, `TextArea`, `SearchField`, `Checkbox`, `CheckboxGroup`, `Radio`, `RadioGroup`, `Switch`
- Feedback and overlays: `InlineAlert`, `ProgressBar`, `ProgressCircle`, `Meter`, `Toast`, `Modal`, `Dialog`, `AlertDialog`, `Popover`, `Tooltip`, `Tray`
- Collections and navigation: `ListBox`, `Picker`, `Menu`, `ActionMenu`, `Tabs`, `ComboBox`, `ListView`, `TableView`, `TreeView`
- Advanced inputs and native additions: `NumberField`, `Slider`, `RangeSlider`, `Calendar`, `RangeCalendar`, `DatePicker`, `DateRangePicker`, `ColorSwatch`, `ColorField`, `Avatar`, `Tag`, `TagGroup`

Component implementations should continue to share the package Provider, theme tokens, Uniwind helpers, primitives, accessibility mappers, and motion utilities.
