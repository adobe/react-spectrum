`svg`, `Heading` (title), and `Content` (body).

### Accessibility

When an IllustratedMessage has a `svg` and a `Heading`, the `svg` needs an `aria-hidden` prop and a `role` prop set to `presentation`.

## Labeling

### Accessibility

The title of a IllustratedMessage is typically provided via its `Heading`.
If a `Heading` isn't specified, the `svg` needs an `aria-label` prop and a `role` prop set to `img`.
