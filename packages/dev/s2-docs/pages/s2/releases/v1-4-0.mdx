# v1.4.0


As always, thanks to all our contributors!

## Changelog

### General Changes
- Add `pressScale` prop documentation - [@reidbarber](https://github.com/reidbarber) - [PR](https://github.com/adobe/react-spectrum/pull/9940)
- Agent skill improvements - [@reidbarber](https://github.com/reidbarber) - [PR](https://github.com/adobe/react-spectrum/pull/9939), [PR](https://github.com/adobe/react-spectrum/pull/10083)
### Button
- Update Button color to improve contrast when static - [@snowystinger](https://github.com/snowystinger) - [PR](https://github.com/adobe/react-spectrum/pull/10069)
### Calendar
- Add support for `selectionMode="multiple" in Calendar - [@devongovett](https://github.com/devongovett) - [PR](https://github.com/adobe/react-spectrum/pull/9948)
- Adds `anchorDate` as an argument to `isDateUnavailable` in RangeCalendar to let users change available dates based on the first date the user picks - [@devongovett](https://github.com/devongovett) - [PR](https://github.com/adobe/react-spectrum/pull/9948)
### Checkbox
- Add support for description and error message - [@devongovett](https://github.com/devongovett) - [PR](https://github.com/adobe/react-spectrum/pull/9877), [@snowystinger](https://github.com/snowystinger) - [PR](https://github.com/adobe/react-spectrum/pull/10012)
### ColorField
- Add `prefix` prop for prepending a non-interactive element inside the field - [@snowystinger](https://github.com/snowystinger) - [PR](https://github.com/adobe/react-spectrum/pull/10014), [PR](https://github.com/adobe/react-spectrum/pull/10060)
### Collections
- Fix handling of falsy keys - [@reidbarber](https://github.com/reidbarber) - [PR](https://github.com/adobe/react-spectrum/pull/9993)
- Allow collection `items` to be non-object values such as strings and numbers - [@devongovett](https://github.com/devongovett) - [PR](https://github.com/adobe/react-spectrum/pull/10047)
### Combobox
- Adjust ComboBox item transition to prevent a background color flash when hovering between items quickly - [@reidbarber](https://github.com/reidbarber) - [PR](https://github.com/adobe/react-spectrum/pull/9953)
- Add `prefix` prop for prepending a non-interactive element inside the field - [@snowystinger](https://github.com/snowystinger) - [PR](https://github.com/adobe/react-spectrum/pull/10014), [PR](https://github.com/adobe/react-spectrum/pull/10060)
### LabeledValue
- Add LabeledValue component - [@yihuiliao](https://github.com/yihuiliao) - [PR](https://github.com/adobe/react-spectrum/pull/9965), [PR](https://github.com/adobe/react-spectrum/pull/10061)
### ListView
- Add drag and drop support - [@LFDanLu](https://github.com/LFDanLu) - [PR](https://github.com/adobe/react-spectrum/pull/9791), [PR](https://github.com/adobe/react-spectrum/pull/10082)
- Ensure ListView renders its divider and borders with the proper colors in HCM - [@LFDanLu](https://github.com/LFDanLu) - [PR](https://github.com/adobe/react-spectrum/pull/10108)
### Menu
- Adjust Menu item transition to prevent a background color flash when hovering between items quickly - [@reidbarber](https://github.com/reidbarber) - [PR](https://github.com/adobe/react-spectrum/pull/9953)
- Add hold affordance icon for `longPress` MenuTrigger variants with ActionButton and ToggleButton - [@LFDanLu](https://github.com/LFDanLu) - [PR](https://github.com/adobe/react-spectrum/pull/10066)
### NumberField
- Add `prefix` prop for prepending a non-interactive element inside the field - [@snowystinger](https://github.com/snowystinger) - [PR](https://github.com/adobe/react-spectrum/pull/10014), [PR](https://github.com/adobe/react-spectrum/pull/10060), [PR](https://github.com/adobe/react-spectrum/pull/10070)
### Picker
