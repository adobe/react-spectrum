# v1.4.0

v1.4.0 is a major feature release. Drag and drop is now fully supported in [ListView](../ListView), [TableView](../TableView), and [TreeView](../TreeView) — with a customizable `DragPreview` component for tailoring the appearance of dragged items. The new [LabeledValue](../LabeledValue) component renders a read-only key-value pair. [TextField](../TextField) gains a `prefix` prop for prepending text or an icon. [Calendar](../Calendar) receives major enhancements: multi-select, week and day views, `weeksInMonth`, and month/year picker navigation. [Checkbox](../Checkbox), [RadioGroup](../RadioGroup), and [Switch](../Switch) now support help text, and [Slider](../Slider) gains a `SliderFill` component for a filled track. TableView also adds highlight selection style and a new `TableFooter` component.

As always, thanks to all our contributors!
