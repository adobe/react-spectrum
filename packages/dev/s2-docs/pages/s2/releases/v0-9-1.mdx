# v0.9.1
<Time date={date} />

Version 0.9.1 is a patch release that fixes focus visible styles in Button and TagGroup, updates ContextualHelp width, and improves Tabs selection indicator behavior.

## Updates
* [Button](../Button.html): Fix focus visible styles from being applied on standard focus
* [ContextualHelp](../ContextualHelp.html): Update width to match Spectrum designs
* [Tabs](../Tabs.html): Update selection indicator when tab text changes
* [TagGroup](../TagGroup.html): Fix focus visible styles from being applied on standard focus
