# v0.11.0

Version 0.11.0 introduces the new SelectBoxGroup component (alpha), fixes ComboBox empty state rendering, improves Picker dropdown styling, and adds support for custom tab layouts with collapse behavior.

## New Components
* [SelectBoxGroup](../SelectBoxGroup)

## Updates
* [ComboBox](../ComboBox): Fix empty state rendering when no items match the current query
* [Picker](../Picker): Fix erroneous dropdown outline when opening the Picker via click
* [Tabs](../Tabs): Support collapse behavior on Tabs when customizing the layout
