# useListBox

<PageDescription>{docs.exports.useListBox.description}</PageDescription>

`ListBox` follows the [Collection Components API](collections?component=ListBox), accepting static collections of `ListBoxItem` elements.

```tsx render files={["starters/hooks/src/ListBox.tsx", "starters/hooks/src/ListBox.css"]}
"use client";
import {ListBox, ListBoxItem} from 'hooks-starter/ListBox';

<ListBox aria-label="Alignment" selectionMode="single">
  <ListBoxItem id="left">Left</ListBoxItem>
  <ListBoxItem id="middle">Middle</ListBoxItem>
  <ListBoxItem id="right">Right</ListBoxItem>
</ListBox>
```
