# useCalendar

<PageDescription>{docs.exports.useCalendar.description}</PageDescription>

A calendar is built from scratch with `useCalendar`, `useCalendarGrid`, and `useCalendarCell`. The previous and next navigation buttons reuse the [Button](../Button) component from React Aria Components.

```tsx render files={["starters/hooks/src/Calendar.tsx", "starters/hooks/src/Calendar.css", "starters/docs/src/Button.tsx", "starters/docs/src/Button.css"]}
"use client";
import {Calendar} from 'hooks-starter/Calendar';

<Calendar aria-label="Event date" />
```
