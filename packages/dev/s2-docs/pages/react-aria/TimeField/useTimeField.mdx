# useTimeField

<PageDescription>{docs.exports.useTimeField.description}</PageDescription>

This example builds a time field from scratch with `useTimeField` and `useDateSegment`. It shares its segment rendering with [useDateField](../DateField/useDateField) — only the state hook differs, using `useTimeFieldState`.

```tsx render files={["starters/hooks/src/TimeField.tsx", "starters/hooks/src/DateField.css", "starters/hooks/src/TimeField.css", "starters/hooks/src/Form.css"]}
"use client";
import {TimeField} from 'hooks-starter/TimeField';

<TimeField label="Event time" />
```
