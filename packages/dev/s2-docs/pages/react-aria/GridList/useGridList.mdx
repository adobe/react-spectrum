# useGridList

<PageDescription>{docs.exports.useGridList.description}</PageDescription>

This example uses `useGridList` for the list and `useGridListItem` for each row, building the collection with the [Collection Components API](collections?component=GridList). Unlike a listbox, each row can contain interactive elements such as the [Button](../Button) reused here.

```tsx render files={["starters/hooks/src/GridList.tsx", "starters/hooks/src/GridList.css", "starters/docs/src/Button.css"]}
"use client";
import {GridList, GridListItem} from 'hooks-starter/GridList';
import {Button} from 'vanilla-starter/Button';

<GridList aria-label="Pokémon" selectionMode="multiple" selectionBehavior="replace">
  <GridListItem id="charizard" textValue="Charizard">
    Charizard
    <Button variant="secondary" onPress={() => alert('Info for Charizard')}>Info</Button>
  </GridListItem>
  <GridListItem id="blastoise" textValue="Blastoise">
    Blastoise
    <Button variant="secondary" onPress={() => alert('Info for Blastoise')}>Info</Button>
  </GridListItem>
  <GridListItem id="venusaur" textValue="Venusaur">
    Venusaur
    <Button variant="secondary" onPress={() => alert('Info for Venusaur')}>Info</Button>
  </GridListItem>
</GridList>
```
