# useColorWheel

<PageDescription>{docs.exports.useColorWheel.description}</PageDescription>

This example builds a color wheel from scratch with `useColorWheel`, which adjusts the hue channel along a circular gradient track. The thumb grows when focused via the keyboard.

```tsx render
"use client";
import React from 'react';
import {useColorWheel} from 'react-aria/useColorWheel';
import {useColorWheelState} from 'react-stately/useColorWheelState';
import {useFocusRing} from 'react-aria/useFocusRing';

const RADIUS = 100;
const TRACK_THICKNESS = 28;
const THUMB_SIZE = 20;

function ColorWheel(props) {
  let state = useColorWheelState(props);
  let inputRef = React.useRef(null);
  let {trackProps, thumbProps, inputProps} =
    useColorWheel({...props, outerRadius: RADIUS, innerRadius: RADIUS - TRACK_THICKNESS}, state, inputRef);
  let {focusProps, isFocusVisible} = useFocusRing();

  return (
    <div style={{position: 'relative', display: 'inline-block'}}>
      <div {...trackProps} style={trackProps.style} />
      <div
        {...thumbProps}
        style={{
          ...thumbProps.style,
          boxSizing: 'border-box',
          width: isFocusVisible ? TRACK_THICKNESS + 4 : THUMB_SIZE,
          height: isFocusVisible ? TRACK_THICKNESS + 4 : THUMB_SIZE,
          borderRadius: '50%',
          border: '2px solid white',
          boxShadow: '0 0 0 1px black, inset 0 0 0 1px black',
          background: state.getDisplayColor().toString('css')
        }}>
        <input ref={inputRef} {...inputProps} {...focusProps} />
      </div>
    </div>
  );
}

<ColorWheel defaultValue="hsl(30, 100%, 50%)" />
```

## Features

There is no native HTML element to implement a color wheel. `useColorWheel` helps achieve an accessible color wheel that can be styled as needed, typically combined with a [color area](../ColorArea/useColorArea) to adjust saturation and brightness.

* Adjusts the hue channel of an HSB color along a circular gradient track
* A visually hidden `<input type="range">` exposes the value to assistive technology
* Support for mouse, touch, and keyboard interactions
* Internationalized hue value announcements

## State management

`useColorWheel` requires a `state` object created with the `useColorWheelState` hook from `react-stately`, which holds the `Color` value, converts between the hue angle and thumb position, and renders the circular gradient track.
