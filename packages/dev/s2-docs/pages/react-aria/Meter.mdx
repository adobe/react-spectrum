## Value

By default, the `value` prop is a percentage between 0 and 100. Use the `minValue`, and `maxValue` props to set a custom value scale.

<VisualExample
  component={VanillaMeter}
  docs={docs.exports.Meter}
  links={docs.links}
  props={['minValue', 'maxValue']}
  initialProps={{
    label: 'Data usage',
    minValue: 0,
    maxValue: 500,
    value: 250,
    formatOptions: {
      style: 'unit',
      unit: 'megabyte'
    }
  }} />
