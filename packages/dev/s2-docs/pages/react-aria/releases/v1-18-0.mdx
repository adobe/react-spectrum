# v1.18.0

v1.18.0 is a feature-packed release. [Calendar](../Calendar) gains multi-select, week and day views, `weeksInMonth`, and month/year picker navigation. The new [`SliderFill`](../Slider) component makes it easy to render a filled track without manual render props, and [`TableFooter`](../Table) adds a standard footer row to Table. Drag and drop is now fully supported across Spectrum 2 ListView, TableView, and TreeView — including a customizable drag preview component. [Checkbox](../Checkbox), [RadioGroup](../RadioGroup), and [Switch](../Switch) now support help text. Collection `items` can now be plain strings and numbers in addition to objects.

As always, thanks to all our contributors!
