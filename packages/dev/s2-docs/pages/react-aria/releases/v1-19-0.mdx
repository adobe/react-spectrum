# v1.19.0

This release adds a new [`keyboardNavigationBehavior`](/react-aria/GridList.html#keyboard-navigation-behavior) prop to [GridList](/react-aria/GridList.html) and [Tree](/react-aria/Tree.html), enabling Tab-key navigation between rows and full interaction with embedded text fields and other focusable elements. [Autocomplete](/react-aria/Autocomplete.html) receives two keyboard fixes: CJK IME input now correctly applies virtual focus to the first matching item, and arrow keys properly move the cursor in inline mode. [Popover](/react-aria/Popover.html) gains a new `getTargetRect` prop for positioning overlays relative to arbitrary rectangles or character positions. Menu's `onAction` callback now provides both the item key and value, and `DragTypes.has()` accepts multiple MIME types and wildcards.
