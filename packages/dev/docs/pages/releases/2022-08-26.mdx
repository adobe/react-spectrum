# August 26, 2022 Release

ðŸš€ We are excited to announce the release of our ListView component!  A ListView displays a list of interactive items, and allows a user to navigate, select, or perform an action. It supports interactive elements within items, multi-selection, row actions, arrow key navigation, disabled items, async loading, infinite scrolling, and more.  Also included in this release is an update to our `FocusScope`, improving how we restore focus when a trigger no longer exists.

Thanks to all our contributors for helping with this release!
