# March 21, 2022 Release

## Fixes
- Updated placeholder color contrast for `Picker` - [@jnurthen](https://github.com/jnurthen) - [PR](https://github.com/adobe/react-spectrum/pull/2760)
- Fix pasting into `TextFields` in `Dialog` on iOS - [@LMulvey](https://github.com/LMulvey) - [PR](https://github.com/adobe/react-spectrum/pull/2939)
- Fix focus ring not disappearing when elements becomes disabled - [@devongovett](https://github.com/devongovett) - [PR](https://github.com/adobe/react-spectrum/pull/2941)


## Under construction

Updates to pre-released components listed below. Please feel free to try them out, and report any issues you encounter.

- Add support for hsl/hsb in ColorArea - [@snowystinger](https://github.com/snowystinger) - [PR](https://github.com/adobe/react-spectrum/pull/2887)
- Add support for unavailable dates in `Calendar` and `DatePicker` - [@razvancir96](https://github.com/razvancir96) - [PR](https://github.com/adobe/react-spectrum/pull/2793)
