## New features
- `useMove` interaction hook - [@mischnic](https://github.com/mischnic) - [PR](https://github.com/adobe/react-spectrum/pull/1106)
- Semantic colors for icons - [@LFDanLu](https://github.com/LFDanLu) -  [PR](https://github.com/adobe/react-spectrum/pull/1182)

## Fixed
- Update label css to stop shifts in position - [@ktabors](https://github.com/ktabors) - [PR](https://github.com/adobe/react-spectrum/pull/984)
- Improve scrolling issues with modals and trays in mobile Safari - [@devongovett](https://github.com/devongovett) - [PR](https://github.com/adobe/react-spectrum/pull/1105)
- Only add `aria-describedby` in `useTooltipTrigger` when open - [@majornista](https://github.com/majornista) - [PR](https://github.com/adobe/react-spectrum/pull/1150)

- Improve clean up of `user-select` on in `usePress` - [@LFDanLu](https://github.com/LFDanLu) - [PR](https://github.com/adobe/react-spectrum/pull/1185)
- Removed `readOnly` from `Slider` - [@mischnic](https://github.com/mischnic) - [PR](https://github.com/adobe/react-spectrum/pull/1083)
- Fix `usePreventScroll` on iOS with React 17 - [@devongovett](https://github.com/devongovett) - [PR](https://github.com/adobe/react-spectrum/pull/1196)
