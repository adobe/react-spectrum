# May 5, 2021 Release

This release we give you plenty of bug fixes, and a new component – **NumberField**! This component is stacked with features and helpful accessibility and internationalization support. NumberField offers internationalized number formatting and parsing, support for the Latin, Arabic, and Han decimal numbering systems, validation, allows incrementing/decrementing via stepper buttons, arrow keys, and scrolling, and much more. See a full list of functionality in our [React Spectrum](https://react-spectrum.adobe.com/react-spectrum/NumberField.html), [React Aria](https://react-spectrum.adobe.com/react-aria/useNumberField.html) and [React Stately](https://react-spectrum.adobe.com/react-stately/useNumberFieldState.html) docs. Also you can learn more about the complexities of NumberField by reading our latest [blog post](https://react-spectrum.adobe.com/blog/how-we-internationalized-our-numberfield.html).

Thank you to all our contributors for their help with this release.
