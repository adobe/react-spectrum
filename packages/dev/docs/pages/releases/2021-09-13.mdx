# September 13, 2021 Release

Today we're excited to announce the stable release of TableView! ðŸš€ðŸš€ This component includes many useful features such as selection, keyboard navigation, sorting, tiered column headers,
async data loading, infinite scrolling, virtualization and type ahead. You can see the `TableView` in action in the [React Spectrum docs](https://react-spectrum.adobe.com/react-spectrum/TableView.html) also view our [React Aria docs](https://react-spectrum.adobe.com/react-aria/useTable.html) for more details on the `useTable` hook.

Also included in this release is a new option for help text included with [Combobox](https://react-spectrum.adobe.com/react-spectrum/ComboBox.html#help-text), [Picker](https://react-spectrum.adobe.com/react-spectrum/Picker.html#help-text), [TextField](https://react-spectrum.adobe.com/react-spectrum/TextField.html#help-text), [TextArea](https://react-spectrum.adobe.com/react-spectrum/TextArea.html#help-text), [SearchField](https://react-spectrum.adobe.com/react-spectrum/SearchField.html#help-text) and [NumberField](https://react-spectrum.adobe.com/react-spectrum/NumberField.html#help-text) via the new `useField` [hook](https://react-spectrum.adobe.com/react-aria/useField.html). This hook associates a form control with a label, and an optional description and/or error message. `useField` takes care of creating ids for each element and associating them with the correct ARIA attributes (`aria-labelledby` and `aria-describedby`). Many React Aria hooks have been updated to support this out of the box as well. See examples in the component docs, [aria examples](https://react-spectrum.adobe.com/react-aria/useTextField.html#example) or the `useField` [hook](https://react-spectrum.adobe.com/react-aria/useField.html).

Plenty of other new features, bug fixes and updates to our documentation in the notes below. Thank you all for your contributions! ðŸ¤©
