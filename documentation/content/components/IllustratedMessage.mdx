import IllustratedMessage from '@react/react-spectrum/IllustratedMessage';

An **IllustratedMessage** displays an illustration icon and a message, usually in an empty state or on an error page. It is also used inside a [DropZone](/components/DropZone).

## Example

```jsx
<IllustratedMessage
  heading="No Results"
  description="Try another search"
  illustration={<svg>...</svg>} />
```

## Internationalization

To internationalize an IllustratedMessage, localized strings should be passed to the `heading` and `description` props.

## Accessibility

IllustratedMessage uses a [Heading](/components/Heading) to wrap the `heading` and a paragraph to wrap the `description`.

The `illustration` is assumed to be a decorative image and `aria-hidden` will be added to the rendered illustration element unless it is otherwise labelled using `aria-label` or `aria-labelledby`.

### `ariaLevel` prop
Use the `ariaLevel` prop to ensure the appropriate heading level within the page hierarchy for accessibility.  The `ariaLevel` will not change the appearance.

```jsx
<IllustratedMessage
  heading="No Results"
  description="Try another search"
  illustration={<svg>...</svg>}
  ariaLevel={3} />
```
