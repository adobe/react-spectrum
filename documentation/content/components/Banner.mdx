import Banner from '@react/react-spectrum/Banner';
import ModalTrigger from '@react/react-spectrum/ModalTrigger';
import Button from '@react/react-spectrum/Button';
import Dialog from '@react/react-spectrum/Dialog';

## Overview

A **banner** is an additional label an existing component may have. Banners cannot be interacted with. Banners in Spectrum have three variations for different uses as well as the ability to place it overlaid in the top-right corner of a container.

## Example

<Banner header="Hello" content="world" />

```jsx
import Banner from '@react/react-spectrum/Banner';

<Banner header="Hello" content="world" />
```

## Variants

There are three banner variants to choose from in Spectrum. The `variant` prop controls the main variant of the banner. The `info` variant is the default one.

### Info

<Banner header="Info" content="A relaxing blue" variant="info" />

```jsx
<Banner header="Info" content="A relaxing blue" variant="info" />
```

### Warning

<Banner header="Warning" content="A worrying orange" variant="warning" />

```jsx
<Banner header="Warning" content="A worrying orange" variant="warning" />
```

### Error

<Banner header="Error" content="An alarming red" variant="error" />

```jsx
<Banner header="Error" content="An alarming red" variant="error" />
```

## Content

Banners can have a header, content, or both. A header can be provided through the `header` prop, and content for below the header can be provided through the `content` prop.

<Banner header="Header only" />
<Banner content="Content only" />
<Banner header="Header here" content="and content over here" />

```jsx
<Banner header="Header only" />
<Banner content="Content only" />
<Banner header="Header here" content="and content over here" />
```

## Corner Placement

In addition to the variant, banners can be placed in the top-right corner of its container by giving them a `corner` prop. Note that the `position` of the containing element needs to be either `relative` or `absolute`.

<div style={{width: '200px', height: '100px', position: 'relative', border: '1px solid black'}}>
  <Banner header="Corner" corner />
</div>

```jsx
<div style={{width: '200px', height: '100px', position: 'relative', border: '1px solid black'}}>
  <Banner header="Corner" corner />
</div>
```

This also works quite well with modal dialogs using the `ModalTrigger` and `Dialog` components!

<ModalTrigger>
  <Button label="Show Modal Dialog with Corner Banner" variant="primary" modaltrigger />
  <Dialog
    modalcontent
    title="Pay Attention To Me"
    confirmLabel="Yep"
    size="S"
    cancelLabel="Nope">
    <Banner header="Corner" corner />
    <span>
    the modal dialog content goes here
    </span>
  </Dialog>
</ModalTrigger>

```jsx
<ModalTrigger>
  <Button label="Show Modal Dialog with Corner Banner" variant="primary" modaltrigger />
  <Dialog
    modalcontent
    title="Pay Attention To Me"
    confirmLabel="Yep"
    size="S"
    cancelLabel="Nope">
    <Banner header="Corner" corner />
    <span>
    the modal dialog content goes here
    </span>
  </Dialog>
</ModalTrigger>
```

## Internationalization

To internationalize a banner, a localized string should be passed to either `header` or `content` props of the banner.

## Accessibility

Banners are accessible by default. The correct aria roles will automatically be applied.
